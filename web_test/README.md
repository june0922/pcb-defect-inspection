# Web Test Directory

본 `web_test` 폴더는 AI 모델의 추론 결과(결함 위치, 신뢰도 등)를 **웹 브라우저 환경에서 시각적으로 확인하고 테스트**하기 위한 웹 기반 뷰어(Web Viewer) 코드 및 리소스가 포함되어 있습니다.

## 주요 구성 요소

* **`index.html`**
  * 리뷰 화면의 UI를 구성하는 메인 HTML 파일입니다.
* **`style.css`**
  * 웹 화면의 레이아웃, 시각적 스타일링 및 결함 박스의 색상 표시(신뢰도 기반) 등을 정의하는 스타일시트입니다.
* **`script.js`**
  * 이미지 로드, 결함 Bounding Box 렌더링, 썸네일 리스트 생성, 키보드 이벤트(판정 처리) 등 웹에서의 동적 상호작용 로직을 담당하는 자바스크립트 파일입니다.
* **`generate_results.py`**
  * 모델 추론 후 도출된 데이터를 웹 화면에서 렌더링하기 적합한 형태(예: JSON 형식 등)로 변환해주는 파이썬 스크립트입니다.
  * 테스트 환경에서 가상의 결과를 생성하거나, 실제 추론 결과 데이터를 파싱하여 웹에 반영할 때 사용됩니다.

## 목적

로컬 GUI 어플리케이션(`app` 폴더)과는 별개로, **웹 환경**에서 검사 결과를 시각화하고 검토 프로세스를 테스트해보기 위한 프로토타입 또는 보조 수단으로 활용됩니다.
