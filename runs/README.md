# runs — 모델 학습 결과 저장소

YOLO26 모델의 다양한 학습 및 하이퍼파라미터 튜닝 결과물(`*.pt` 가중치 파일, 학습 로그, 분석 차트 등)이 자동으로 생성되어 저장되는 디렉토리입니다.

## 주요 하위 폴더 구성

- `kfold/`: `train_kfold.py` (K-Fold 교차 검증 학습) 실행 결과가 각 fold별로 저장됩니다.
- `tune/`: `tune.py` (하이퍼파라미터 튜닝) 실행 결과가 저장됩니다.
- `kfold_tune/`: `train_kfold_tune.py` (K-Fold 기반 튜닝) 실행 결과가 각 fold별로 저장됩니다.

## 주의 사항
- 이 폴더 하위에 생성되는 대용량 학습 결과 파일들은 필요 시 최종 선별하여 배포 위치(`weights/` 또는 `app_front/models/`)로 복사하여 사용합니다.
