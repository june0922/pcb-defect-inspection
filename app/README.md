# PCB Defect Inspection Review Station - 사용자 매뉴얼

본 매뉴얼은 AI 모델(YOLOv8 Nano 5-Fold 앙상블)이 탐지한 PCB 결함을 작업자가 최종적으로 검토하고 판정하기 위한 **Review Station(리뷰 스테이션) 프로그램**의 사용 방법을 안내합니다.
내부적으로 5개의 가중치 모델을 앙상블(WBF)하여 추론을 수행하며, GUI 기반으로 작업자가 피로도 없이 빠르고 정밀하게 검토할 수 있도록 동작합니다.

---

## 1. 프로그램 개요
이 프로그램은 AI가 1차적으로 찾아낸 결함 후보들을 작업자가 직관적으로 검토 및 판정할 수 있도록 설계된 전문 검사 툴입니다. 
AI의 판단 신뢰도(Confidence Score)에 따라 결함이 시각적으로 분류되며, 키보드 전역 단축키를 이용해 초고속 라벨링과 뷰어 제어가 가능합니다.

---

## 2. 시작하기

1. **프로그램 실행:** `app/run.py`를 실행하여 프로그램을 켭니다.
2. **폴더 열기:** 
   - 좌측 상단 메뉴 바에서 **[File] > [Open Folder...]** 를 클릭합니다.
   - 검사할 이미지가 들어있는 폴더를 선택합니다.
3. **AI 연산 대기:** 
   - 폴더를 선택하면 화면 하단 상태 표시줄에 진행률 바가 나타나며 AI 모델 로딩 및 앙상블 추론 연산이 백그라운드에서 시작됩니다.
   - 연산이 완료된 이미지는 즉시 화면에 렌더링됩니다.

---

## 3. 화면 구성 및 조작 가이드

프로그램 화면은 효율적인 듀얼 뷰어와 썸네일 리스트로 구성됩니다.

### 🌍 Global View (좌측 메인 화면)
- 현재 선택된 전체 이미지를 미니맵 형태로 보여줍니다.
- AI가 찾아낸 모든 결함 위치가 표시되며, 현재 포커스된 결함은 하늘색 십자선(Crosshair)으로 강조됩니다.
- 우측 화면에서 마우스를 조작하거나 편집할 때, 브러쉬 위치와 편집 내용이 실시간으로 동기화됩니다.

### 🔍 Local View (우측 확대 화면 - VisionViewer)
- 고배율로 이미지를 자유롭게 확대/축소하고 패닝할 수 있는 대화형 뷰어입니다.
- **마우스 조작:**
  - **우클릭 드래그:** 이미지를 상하좌우로 자유롭게 이동(Panning)합니다.
  - **마우스 휠:** 마우스 포인터를 중심으로 확대/축소(Zoom)합니다.
  - **좌클릭 드래그 (브러쉬 마스킹):** 마우스로 문질러 이미지 위의 결함 부위를 수동으로 가리거나 지울 수 있습니다. (RGB: 255 흰색 마스킹)

### 🎞️ Thumbnail Strip (하단 썸네일 리스트)
- AI가 찾아낸 결함들을 96x96 썸네일로 나열합니다. 
- 이미지 찌그러짐을 방지하기 위해 회색 여백(Padding)이 적용되어 원본 비율 그대로 표시됩니다.
- 썸네일 테두리 색상으로 판정 상태를 바로 알 수 있습니다.
  - 🔘 **회색:** 아직 판정하지 않은 대기 상태 (Pending)
  - 🔴 **빨간색:** 실제 결함 (Fail)
  - 🟢 **초록색:** 가짜 결함/정상 (Pass)

---

## 4. 시각적 색상 가이드 (결함 박스)

AI의 확신도(Confidence Score)에 따라 결함 코너 브라켓의 색상이 다릅니다.
* 🔴 **빨간색 (90% 이상):** 확실한 불량
* 🟡 **노란색 (70% ~ 89%):** 주의 요망 / 모호함
* 🔘 **회색 (70% 미만):** 가짜 결함 의심

> **Tip:** `Shift` 키를 꾹 누르고 있으면 화면의 여러 결함 박스 중 **현재 썸네일에서 선택된 결함 박스만 남기고 나머지 박스들은 투명하게 숨겨집니다.** 겹쳐있는 결함을 구별할 때 유용합니다.

---

## 5. 마우스 및 키보드 조작 가이드

마우스와 키보드를 함께 사용하면 작업 속도가 비약적으로 향상됩니다. 키보드 단축키는 현재 포커스에 상관없이 앱 어디서든 작동합니다.

### 🖱️ 마우스 조작 (Vision Viewer 전용)
- **좌클릭 드래그 (브러쉬 마스킹):** 마우스로 문질러 이미지 위의 결함 부위를 수동으로 흰색(255)으로 덮어서 가리거나 지웁니다.
- **우클릭 드래그 (패닝):** 이미지를 잡고 상하좌우로 자유롭게 화면을 이동시킵니다.
- **마우스 휠:** 마우스 포인터가 위치한 곳을 중심으로 이미지를 확대하거나 축소합니다.

### 🏷️ 결함 판정 (Labeling)
- **`1` ~ `6` 숫자키:** 해당 결함을 특정 불량 유형(Fail 1~6)으로 판정합니다.
- **`Space` 키:** 해당 결함을 가짜 결함 / 정상(Pass)으로 판정합니다.
  > *판정을 내리면 자동으로 다음 대기(Pending) 상태인 썸네일로 이동합니다.*
- **`←` / `→` 방향키:** 수동으로 이전/다음 썸네일로 이동합니다.

### 👁️ 뷰어 제어 (Navigation)
- **`W`, `A`, `S`, `D`:** 상/좌/하/우 패닝 (키보드로 화면 이동)
- **`Q` / `E`:** 축소(Q) / 확대(E)
- **`Shift` (Hold):** 현재 선택된 결함 박스 이외의 모든 오버레이 일시 숨김

### 🖌️ 브러쉬 편집 및 연산
- **`-` / `=` 키:** 수동 마스킹용 브러쉬 크기 축소 / 확대 (1px ~ 100px)
- **`F5` 키:** 브러쉬 마스킹 편집 후 **현재 이미지에 대해서만 AI 추론을 다시 실행**합니다. 수동으로 가려진 부위는 검출되지 않습니다.
- **`Ctrl + S`:** 현재까지의 라벨링 리뷰 결과를 즉시 JSON 파일로 저장합니다.

---

## 6. 옵션 및 환경 설정 (Settings)

상단 메뉴에서 **[Option] > [Settings...]** 를 클릭하여 AI 탐지 민감도를 조절할 수 있습니다.

* **Min Confidence (최소 신뢰도 - 기본값 40):** 이 수치보다 낮은 결함은 무시합니다. (먼지가 많이 잡히면 수치를 올리세요)
* **Max Confidence (최대 신뢰도 - 기본값 90):** 이 수치보다 높은 결함은 명백한 불량으로 간주, 리뷰에서 제외시킬 수 있습니다.
* **IoU Threshold (기본값 0.45):** 겹침 허용치 (NMS/WBF 최적화 튜닝)

> ⚠️ **주의:** 설정을 변경하고 [Ok] 버튼을 누르면 AI 연산을 다시 수행합니다. 이때 수동으로 브러쉬 마스킹(편집)을 해둔 이미지들의 **편집 내역은 삭제되지 않고 안전하게 보존(캐시)**되어 그 위에서 다시 연산됩니다.

---

## 7. 판정 결과 저장하기 (`Ctrl + S`)

리뷰 중 언제든지 **`Ctrl + S`** 를 누르거나 상단 메뉴에서 **[File] > [Save Review Results]** 를 클릭해 데이터를 저장할 수 있습니다.
- 폴더 내에 `review_results.json` 파일이 생성되며, 어떤 이미지가 어떤 유형의 결함으로 처리되었는지 최종 라벨링 데이터가 남게 됩니다.
